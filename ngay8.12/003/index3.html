setFullYear() set thời gian theo năm (yyyy) setMonth() set thời gian theo tháng (0-11) setDate() set thời gian theo ngày (1-31) setHours() set thời gian theo giò (0-23) setMinutes() set thời gian theo phút (0-59) setSeconds() set thời gian theo giây (0-59) setTime() set thời gian theo timestamp (milliseconds since January 1, 1970)
