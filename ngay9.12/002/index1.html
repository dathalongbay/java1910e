Kế thừa trong class javascript
        ví dụ : ClassCon extends ClassCha
        trong constructor của class Con mà muốn gọi đến construct của class Cha
        thì sẽ sử dụng từ khóa super()
        => Khi 1 class con kế thừa từ class cha
        thì class con có thể gọi đến tất cả các thuộc tính cũng như các phương thức
        của class cha
        => thuộc tính class chỉ đặc điểm tính chất
        ví dụ tên , tuổi , quê quán của sinh viên
        => phương thức trong class chỉ ra 1 hành động cụ thể của class
        ví dụ class sinhvien có các phương thức dihoc() , thi() , totnghiep()
